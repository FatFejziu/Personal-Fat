# My Super Advanced Python Project

## Features
- User and Post Management
- API support
- Database interaction (SQLite)
- Logging and error handling
- Unit Testing

## Installation
1. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

2. Run the app:
    ```bash
    python app/main.py
    ```

## Testing
To run tests:
```bash
python -m unittest discover
